# BlockChain Economic Model

## Core

区块链经济学的本质是，**用经济激励替代信任，通过机制设计让自利行为产生公共利益。** 它是维持 BlockChain 长治久安的设计基础，更是安全与长期繁荣的机制保障。下面具体的理解一下区块链经济学的主要内容。

区块链经济学主要包括一下五个方面：
1. 激励相容（Incentive Compatibility）：让每个参与者追求自身利益最大化的行为，客观上有利于整个网络。
2. 价值捕获（Value Capture）：网络的使用如何转化为代币的价值。
3. 可持续性（Sustainability）：网络能否长期运行，不依赖外部补贴。
4. 安全性保障（Security）：经济激励如何保护网络免受攻击。
5. 网络效应（Network Effects）：如何形成正向循环，让网络价值随用户增长而指数级增长。
